# SOX連動度ダッシュボード

前日のフィラデルフィア半導体株指数（SOX）のリターンと、当日の日本の半導体関連株リターンの相関をまとめて比較するStreamlitアプリです。

## 主な機能

- ユーザー指定の半導体関連124銘柄を登録
- 1か月・3か月・6か月・1年から分析期間を選択
- 注目20銘柄、分野別、個別選択、全銘柄の一括分析
- 会社名、証券コード、分野、関連領域、期間相関、最新20日相関を一覧表示
- ランキングの会社名・コード・分野検索とCSV保存
- 各銘柄について直近20観測日の騰落率、対応SOX日、20日ローリング相関を表示
- 日米の土日・祝日差を考慮し、前日のSOXと当日の日本株を対応
- 分析結果をセッション内に保持し、銘柄詳細の切り替え時に再計算不要
- 20銘柄ずつ分割取得し、多数銘柄の一括分析を安定化
- iPhoneを含むスマートフォン向けレスポンシブUI

## 画面構成

- **相関分析**: 分析期間と銘柄を選び、ランキングと詳細を表示
- **登録銘柄一覧**: 124銘柄を会社名・コード・分野・関連領域で検索
- **ランキング**: 相関上位順の一覧、絞り込み、CSV保存
- **銘柄詳細**: 1社ずつ切り替えて、騰落率チャート・相関推移・20日データを確認
- **取得状況**: 取得成功率と取得できなかった銘柄を確認

## ローカル実行

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## 注意

- 相関は因果関係や将来の値動きを保証しません。
- データはYahoo Finance経由で取得するため、遅延・欠損・一時的な取得失敗が起こる場合があります。
- 全124銘柄の分析は複数回に分けて取得するため、注目20銘柄より時間がかかります。
- 上場期間が短い銘柄やデータが不足する銘柄は、選択期間によって分析対象から除外される場合があります。

# SOX連動度ダッシュボード（統合修正版）

`sox-correlation-app-updated` で追加した機能を維持したまま、ユーザー指定の半導体関連124銘柄を追加した統合版です。

## 維持・統合した機能

- モダンでスマートフォン向けのレスポンシブUI
- 1か月・3か月・6か月・1年の期間選択
- 複数銘柄の一括分析
- 注目20銘柄、分野別、個別選択、全124銘柄の選択方式
- 会社名、証券コード、分野、関連領域を含むランキング
- 登録124銘柄の検索・絞り込み一覧
- 期間相関と最新20日ローリング相関
- 直近20観測日の騰落率、対応SOX取引日、日ごとの20日ローリング相関
- 騰落率チャート、ローリング相関チャート、CSV保存
- 分析結果のセッション保持
- 20銘柄単位の分割取得と取得進捗表示
- 日米の土日・祝日差を考慮した前日SOXとの対応

## ローカル実行

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Streamlit Community Cloudへの反映

GitHubリポジトリ直下の `streamlit_app.py` をこの統合修正版で置き換えてコミットしてください。公開中のアプリへ自動的に反映されます。

## 注意

- 相関は因果関係や将来の値動きを保証しません。
- データはYahoo Finance経由で取得するため、遅延・欠損・一時的な取得失敗が起こる場合があります。
- 全124銘柄の分析は時間がかかるため、普段は「注目20銘柄」または分野別分析が快適です。
